create(options, clientRepresentation)

Creates a new keycloak client.
Parameters:
Name Type Argument Description
options object Request options
Properties
Name Type Argument Default Description
endpoint string The API endpoint, e.g. http://localhost:8080/auth/realms/master/clients-registrations
accessToken string The Initial access token
provider string <optional>
 default The Keycloak client registration provider to use. Valid values are:
- 'default': This is the default value and will use Keycloak client representations
- 'openid-connect': Uses OpenID Connect Dynamic Client registration
- 'saml2-entity-descriptor': Creates clients using SAML2 entity descriptors as the client representation.
clientRepresentation object <optional>
 An object representing the client
Properties
Name Type Argument Description
clientId string <optional>
 The ID of the client to be created when using the 'default' provider
Returns:
A promise that will resolve with the client object
